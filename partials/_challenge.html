A crypto-currency fundraising

Because supporting tech is better with tech

  • Sponsoring is done by sending crypto-coins

    Funds are collected in Ether

    Like in a traditional fundraising, funds will be collected for BeCode in exchange for Thomas' sporting feat. The exciting stuff is that these funds will be collected in Ether, a crypto-currency similar to Bitcoin

  • Funds are publicly visible but safely locked

    Secured in a smart contract

    On the Ethereum network where they are stored, the funds can be safely consulted by all. It's just like a public ledger. The funds are locked in a snippet of code that is called a smart contract: a kind of contract that is responsible for its own execution

  • Only one key to release the funds to BeCode

    The trigger: running 60km

    The funds will be wired to BeCode only if Thomas records a 60km run through the Strava running application, to which the smart contract is connected. Otherwise, the donors will get back their funds
